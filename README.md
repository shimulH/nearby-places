# Nearby Places

Foobar is a Python library for dealing with word pluralization.

## Stack

- React.js
- Redux (Saga)
- Material-UI


## Installation

Setup:

1. `yarn/yarn install`


To run:

1. `yarn start`
2. Open [http://localhost:3000](http://localhost:3000)
